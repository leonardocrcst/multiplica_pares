# Multiplica pares

### Multiplica um valor base pelos números pares dentro de um intervalo

Esse projeto faz parte dos meus estudos e exercícios de exploração de Vue.js

## Versões

0.1 - Multiplica os valores definidos via componente

## Instalação
```
yarn install
```

### Execução
```
yarn serve
```

### Construção
```
yarn build
```

### Lint para o código
```
yarn lint
```
